Actuellement, notre composant nous permettant d'afficher la liste des cryptos est plutôt moche. Nous allons la rendre plus belles et afficher les images correspondantes pour chaque monnaie.

Créons un nouveau composant que nous nommerons <Fichier name="ImageBlock"/>. Ce composant permettra d'afficher les images et les noms de nos Cryptos.

```jsx:ImageBlock.jsx
const ImageBlock = ({ children, crypto }) => {
    return (
        <button className="imageWrapper">
            <img src={crypto.link} alt={crypto.currency_long}/>
            {children}
        </button>
    );
};
  
export default ImageBlock;
```
